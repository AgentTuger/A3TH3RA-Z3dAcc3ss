Rapport AETHERA – 5 Cas D Inexpliqués en France

Date : 13 avril 2025

Classification : Catégorie D – Phénomènes non identifiés après enquête

1. Observation à La Forêt-Fouesnant (Finistère) – Décembre 2018

Description : Une témoin observe une barre jaune avec un centre lumineux vert se déplaçant lentement et silencieusement dans le ciel matinal. Aucun bruit, aucune traînée visible. Phénomène observé pendant plusieurs secondes.

Enquête : Analyses météorologiques, aéronautiques et astronomiques effectuées. Aucune correspondance trouvée.

Conclusion : Cas classé D. Phénomène inexpliqué.

2. Observation à Toulouse (Haute-Garonne) – Date non précisée

Description : Plusieurs témoins signalent simultanément la présence de traits noirs verticaux fixes dans le ciel diurne. Les objets ne semblent pas bouger. Aucune projection d’ombre, aucune distorsion visible. Durée de l’observation : environ 10 minutes.

Enquête : Aucune explication météorologique, aérienne ou optique ne permet de conclure à un phénomène connu.

Conclusion : Cas classé D.

3. Observation à Ecquevilly (Yvelines) – 1993

Description : Un témoin unique observe un phénomène lumineux intense dans le ciel nocturne. Aucun mouvement apparent, forme non définie, lumière blanche légèrement bleutée.

Enquête : Les investigations locales et les recoupements avec le trafic aérien n’ont pas permis de trouver une origine rationnelle.

Conclusion : Cas classé D.

4. Observation à Forbach (Moselle) – 1994

Description : Un objet lumineux a été vu se déplaçant de manière erratique dans le ciel. Il changeait brusquement de direction, avec des accélérations soudaines. Aucun bruit signalé.

Enquête : Absence de corrélation avec des exercices militaires, aéronefs ou drones civils.

Conclusion : Cas classé D.

5. Observation à Nancy (Meurthe-et-Moselle) – 1995

Description : Plusieurs témoins rapportent un objet lent, survolant la ville, doté de lumières clignotantes rouges et blanches. Le déplacement semblait trop lent pour un avion conventionnel.

Enquête : Aucune trace radar et pas de lien avec les avions civils ou militaires en activité.

Conclusion : Cas classé D.
